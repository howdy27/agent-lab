# 요구사항 수행 내역서

## 1단계 — 기본 보안 및 네트워크 설정

### SSH 포트 변경 및 Root 차단
```bash
sudo vi /etc/ssh/sshd_config
# Port 20022
# PermitRootLogin no
```

### 확인
```bash
sudo grep -E "^Port|^PermitRootLogin" /etc/ssh/sshd_config
# Port 20022
# PermitRootLogin no

sudo ss -tulnp | grep sshd
# 0.0.0.0:20022 LISTEN
```

### UFW 방화벽 설정
```bash
sudo ufw enable
sudo ufw allow 20022/tcp
sudo ufw allow 15034/tcp
```

### 확인
```bash
sudo ufw status
# 20022/tcp ALLOW Anywhere
# 15034/tcp ALLOW Anywhere
```


## 2. 계정/그룹/권한 체계

### 그룹 생성
```bash
sudo groupadd agent-common
sudo groupadd agent-core
```

### 계정 생성
```bash
sudo useradd -m -s /bin/bash agent-admin
sudo useradd -m -s /bin/bash agent-dev
sudo useradd -m -s /bin/bash agent-test
```

### 그룹에 계정 추가
```bash
sudo usermod -aG agent-common agent-admin
sudo usermod -aG agent-common agent-dev
sudo usermod -aG agent-common agent-test
sudo usermod -aG agent-core agent-admin
sudo usermod -aG agent-core agent-dev
```

### 확인
```bash
kmoru0103@kmoru0103:~$ id agent-admin
uid=1001(agent-admin) gid=1003(agent-admin) groups=1003(agent-admin),1001(agent-common),1002(agent-core)
kmoru0103@kmoru0103:~$ id agent-dev
uid=1002(agent-dev) gid=1004(agent-dev) groups=1004(agent-dev),1001(agent-common),1002(agent-core)
kmoru0103@kmoru0103:~$ id agent-test
uid=1003(agent-test) gid=1005(agent-test) groups=1005(agent-test),1001(agent-common)
```

### 디렉토리/권한 설정
```bash
sudo chown agent-admin:agent-common /home/agent-admin/agent-app/upload_files
sudo chmod 770 /home/agent-admin/agent-app/upload_files
sudo chown agent-admin:agent-core /home/agent-admin/agent-app/api_keys
sudo chmod 770 /home/agent-admin/agent-app/api_keys
sudo chown agent-admin:agent-core /var/log/agent-app
sudo chmod 770 /var/log/agent-app
sudo setfacl -m g:agent-common:rwx /home/agent-admin/agent-app/upload_files
sudo setfacl -m g:agent-common:--- /home/agent-admin/agent-app/api_keys
sudo setfacl -m g:agent-core:rwx /var/log/agent-app
```

### 확인
